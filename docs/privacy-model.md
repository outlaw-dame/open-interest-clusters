# Privacy Model

Interest-cluster datasets are designed for privacy-respecting discovery.

## Supported exclusion controls

- `discoverable = false`
- `indexable = false`
- profile or post opt-out markers such as:
  - NoAI
  - NoBot
  - NoBots
  - NoCrawl
  - NoCrawling
  - NoIndex
  - NoIndexing
  - NoSearch
  - NoScrape
  - NoScraping
  - NoLLM
  - NoLLMs
  - Robotxt
